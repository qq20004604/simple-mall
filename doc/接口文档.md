# 注册

> /register

请求

```json
{
    "tel": "12345678901",    // 手机号码
    "username": "",    // 用户名 2~20
    "pw": "",        // 密码 8~40位
    "verifyCode": ""        // 验证码
}
```


返回

````json
{
    "code": 200,    // 200正常，0不正常
    "msg"：”“，    // 错误信息
    "data": {
        "username":""    // 正确的话返回用户名
    }
}
````
